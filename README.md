# The game
## Описание игры
Это пошаговая стратегия в которой 2 игрокам на квадратном поле нужно уничтожить героя соперника. В игре есть 2 расы: 
__орки__ и __люди__. \
У игроков есть армия а также по одному ультимативному юниту _(Герою)_, который настраивается до игры.
В процессе игры можно заказывать войска 3 типов: _пехота_, _маги_, _наездники_.
## Запуск
Для компиляции требуется предустановленные компилятор `MinGW`, с прописанным путём.
\
Необходимо запустить скрипт `compile.bat`, который скомпилирует игру.
\
Затем для начала игры запустить `execute.bat`. Для тестов `test_all.bat`.
## Процесс игры и игровые настройки.
Игра рассчитана на 2 человек: первые - орки, вторые - люди. Первые сверху.
\
Управление: выбрать юнита - ЛКМ. Отдать приказ на движени или атаку - ПКМ(двигаться можно только на 1 клетку).
\
Треугольные юниты: маги, квадратные: пехота, круглые: наездники. Шестиугольник: герой.
\
Для выбора героя в аргументах при запуске `execute.bat` нужно указать аргументы: первый для первого игрока, второй для второго.
Варианты: `Wise` и `Mighty`.
\
Для настройки стартовой армии необходимо заполнить файл `field.txt`. Юниты в верхней половине: первого игрока, во нижней: второго. Посередине юнитов
быть не должно. Обозначения: H-Horseman, M-Mage, I-infantry. Звездой обозначаем героя.
\
__Важно!__ Поле обязательно должно иметь размер 20x15. Должен быть ровно 1 герой с каждой стороны. 
\
Игра завершается со смертью одного из героев.


## Структура проекта
* В каталоге `lib` хранятся необходимые библиотеки.
* В каталоге `code` расположен сам код проекта. 
* В разных папках `dirname` хранятся разные части игры.
  * В каждой `dirname` есть папка `src` с исходниками этой части.
  * Также есть папка `include` для хедеров.
  * Иногда, когда есть последующее деление кода по смыслу он разбивается по папкам с соответствующими названиями, структура одинакова в `src` и `include`
* Есть папка `tests`, здесь есть различные исходники для тестирования разных модулей. 
* Остальные директории вспомогательные.
